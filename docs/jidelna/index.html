Soukromí aplikace<\h1> Co Bude k Obědu neshromažďuje žádné tvé osobní ani jiné údaje. K žádným osobním údajům nemá přístup a ani o ně nemá zájem. Aplikace může zísávat pouze anonymní analytická data, pouze pro účel dalšího vývoje a opravy chyb. Tyhle data nijak nelze spojit s tebou ani jinou konkrétní osobou. Jednoduše se o svoje soukromí nemusíš bát.
